⚙️ Email Notification Setup (EmailJS)
📧 One-time setup required — Once configured, employees will automatically notify ytkang@jnfgroup.net when submitting leave requests.
Step 1 — Create Free EmailJS Account
1. Go to emailjs.com → Sign Up Free
2. Email Services → Add New Service → Select Gmail
3. Connect Gmail → Copy the Service ID (e.g. service_abc123)
Step 2 — Create Email Template
1. Email Templates → Create New Template
2. To Email: ytkang@jnfgroup.net
3. Subject: [Leave Request] {{from_name}}
4. Content: {{message}}
5. Save → Copy the Template ID (e.g. template_xyz789)
